# プライバシーポリシー

**最終更新日**: 2025年1月6日

Clevique（以下「当社」）は、BarNoR（バーノア）アプリケーション（以下「本アプリ」）をご利用いただくユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用・保護するかについて説明するものです。

---

## 1. 収集する情報

本アプリは、以下の情報を収集・使用します。すべての情報は**お客様のデバイス内にのみ保存され、当社のサーバーには一切送信されません**。

### 1.1 ユーザーが提供する情報

- **Notion Integration Token**
  - 用途: Notion APIとの連携のため、お客様が任意で入力するトークン
  - 保存場所: デバイス内（暗号化ストレージMMKV）
  - 第三者提供: なし

- **Notionデータベース設定**
  - 用途: スキャンしたデータを保存するNotionデータベースの識別情報
  - 保存場所: デバイス内（暗号化ストレージMMKV）
  - 第三者提供: なし

### 1.2 アプリが自動的に収集する情報

- **スキャン履歴**
  - 用途: スキャンしたバーコード・書籍情報の履歴表示
  - 保存場所: デバイス内（暗号化ストレージMMKV）
  - 第三者提供: なし
  - 削除方法: アプリ内の「履歴をクリア」機能で削除可能

- **カメラアクセス**
  - 用途: バーコードのスキャンのみ
  - 保存: 撮影画像は保存されません（バーコード情報のみ読み取り）
  - 権限: 初回起動時にカメラアクセス許可を要求

### 1.3 収集しない情報

本アプリは以下の情報を**収集しません**：

- 個人を特定できる情報（名前、メールアドレス、電話番号など）
- 位置情報
- 連絡先
- 写真・動画（カメラはバーコード読み取りのみ使用）
- デバイスID・広告ID
- 利用状況の分析データ

---

## 2. 情報の使用目的

収集した情報は、以下の目的でのみ使用されます：

1. **Notion APIとの連携**
   - お客様が提供したIntegration Tokenを使用し、Notion APIに接続
   - スキャンした書籍・商品情報をお客様のNotionデータベースに保存

2. **アプリ機能の提供**
   - バーコードスキャン機能の実行
   - スキャン履歴の表示
   - パッケージ管理機能の提供

3. **ユーザー体験の向上**
   - アプリ設定の保存（テーマ設定など）
   - エラーハンドリング・リトライ機能

---

## 3. 第三者サービスとの連携

本アプリは、機能提供のために以下の第三者サービスと連携します：

### 3.1 Notion API
- **提供元**: Notion Labs, Inc.
- **用途**: お客様のNotionワークスペースへのデータ保存
- **送信データ**: スキャンした書籍・商品情報（タイトル、著者、ISBN、画像URLなど）
- **プライバシーポリシー**: https://www.notion.so/Privacy-Policy-3468d120cf614d4c9014c09f6adc9091

**重要**: Notion APIへのアクセスは、お客様が明示的に提供したIntegration Tokenを使用します。お客様の許可なくデータが送信されることはありません。

### 3.2 OpenBD API
- **提供元**: 株式会社カーリル（日本の書籍情報API）
- **用途**: ISBNコードから書籍情報を取得
- **送信データ**: ISBNコード（バーコード番号）のみ
- **プライバシーポリシー**: https://openbd.jp/

**注意**: ISBNコードは個人情報ではなく、書籍を識別するための公開情報です。

### 3.3 国立国会図書館API（書影取得）
- **提供元**: 国立国会図書館
- **用途**: 書籍の表紙画像URLを取得
- **送信データ**: ISBNコード（バーコード番号）のみ

## 4. データの保存と保護

### 4.1 ローカル保存

すべてのデータは**お客様のデバイス内にのみ保存**され、当社のサーバーには一切送信されません。

- **暗号化**: Notion Integration Token、パッケージ設定、スキャン履歴は、MMKV暗号化ストレージに保存
- **アクセス制限**: 本アプリのみがアクセス可能
- **バックアップ**: iCloudバックアップに含まれる場合があります

### 4.2 セキュリティ対策

- AES暗号化によるデータ保護
- HTTPS通信の使用（Notion API・OpenBD API）
- コード難読化・署名検証

### 4.3 データの削除

- **アプリアンインストール**: すべてのデータが完全に削除されます
- **履歴のクリア**: アプリ内の「履歴をクリア」機能で削除可能
- **トークンの削除**: 設定画面から認証情報をクリア可能

---

## 5. データの第三者提供

当社は、お客様の情報を第三者に販売、交換、譲渡することは**一切ありません**。

ただし、以下の場合を除きます：

1. **お客様の明示的な同意がある場合**
   - Notion APIへのデータ保存（お客様が提供したトークンを使用）

2. **法的要請がある場合**
   - 裁判所命令、法的手続き、政府機関の要請

本アプリは13歳未満のお子様を対象としていません。13歳未満のお子様から意図的に個人情報を収集することはありません。

保護者の方で、お子様が個人情報を提供したと思われる場合は、info@clevique.app までご連絡ください。

## 7. カリフォルニア州居住者の権利（CCPA）

カリフォルニア州消費者プライバシー法（CCPA）に基づき、カリフォルニア州居住者には以下の権利があります：

- **アクセス権**: 収集された個人情報の開示を要求する権利
- **削除権**: 個人情報の削除を要求する権利
- **販売のオプトアウト**: 個人情報の販売を拒否する権利（当社は販売しません）

**注意**: 本アプリは個人情報をサーバーに送信しないため、当社が保有するお客様の個人情報はありません。

## 8. 欧州居住者の権利（GDPR）

EU一般データ保護規則（GDPR）に基づき、欧州経済領域（EEA）居住者には以下の権利があります：

- **アクセス権**: 個人データのコピーを要求する権利
- **訂正権**: 不正確な個人データの訂正を要求する権利
- **削除権（忘れられる権利）**: 個人データの削除を要求する権利
- **処理の制限**: 個人データの処理を制限する権利
- **データポータビリティ**: 個人データの移行を要求する権利
- **異議申立権**: 個人データの処理に異議を申し立てる権利

**注意**: 本アプリは個人データをサーバーに送信しないため、当社が保有するお客様の個人データはありません。すべてのデータはデバイス内に保存され、アプリアンインストールで完全に削除されます。
