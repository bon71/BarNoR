# Privacy Policy

**Last Updated**: January 6, 2025

Clevique ("we," "our," or "us") respects the privacy of users of the BarNoR application (the "App") and is committed to protecting personal information. This Privacy Policy explains what information the App collects, how it is used, and how it is protected.

---

## 1. Information We Collect

The App collects and uses the following information. **All information is stored only on your device and is never sent to our servers.**

### 1.1 Information You Provide

- **Notion Integration Token**
  - Purpose: To connect with Notion API, optionally provided by you
  - Storage: On-device (encrypted MMKV storage)
  - Third-party sharing: None

- **Notion Database Configuration**
  - Purpose: Identification information for the Notion database where scanned data is saved
  - Storage: On-device (encrypted MMKV storage)
  - Third-party sharing: None

### 1.2 Information Automatically Collected

- **Scan History**
  - Purpose: Display history of scanned barcodes and book information
  - Storage: On-device (encrypted MMKV storage)
  - Third-party sharing: None
  - Deletion: Can be deleted using the "Clear History" feature in the app

- **Camera Access**
  - Purpose: Barcode scanning only
  - Storage: Captured images are not saved (only barcode information is read)
  - Permission: Camera access permission requested on first launch

### 1.3 Information We Do Not Collect

The App **does not collect** the following information:

- Personally identifiable information (name, email, phone number, etc.)
- Location data
- Contacts
- Photos/videos (camera used for barcode scanning only)
- Device ID/Advertising ID
- Usage analytics data

---

## 2. How We Use Information

Collected information is used only for the following purposes:

1. **Notion API Integration**
   - Connect to Notion API using the Integration Token you provide
   - Save scanned book/product information to your Notion database

2. **App Functionality**
   - Execute barcode scanning
   - Display scan history
   - Provide package management features

3. **User Experience Enhancement**
   - Save app settings (theme settings, etc.)
   - Error handling and retry functionality

---

## 3. Third-Party Service Integration

The App integrates with the following third-party services to provide functionality:

### 3.1 Notion API
- **Provider**: Notion Labs, Inc.
- **Purpose**: Save data to your Notion workspace
- **Data Sent**: Scanned book/product information (title, author, ISBN, image URL, etc.)
- **Privacy Policy**: https://www.notion.so/Privacy-Policy-3468d120cf614d4c9014c09f6adc9091

**Important**: Access to Notion API uses the Integration Token you explicitly provide. No data is sent without your permission.

### 3.2 OpenBD API
- **Provider**: Calil Inc. (Japanese book information API)
- **Purpose**: Retrieve book information from ISBN codes
- **Data Sent**: ISBN code (barcode number) only
- **Privacy Policy**: https://openbd.jp/

**Note**: ISBN codes are not personal information; they are public identifiers for books.

### 3.3 National Diet Library API (Book Cover Images)
- **Provider**: National Diet Library of Japan
- **Purpose**: Retrieve book cover image URLs
- **Data Sent**: ISBN code (barcode number) only

---

## 4. Data Storage and Protection

### 4.1 Local Storage

All data is **stored only on your device** and is never sent to our servers.

- **Encryption**: Notion Integration Token, package settings, and scan history are stored in encrypted MMKV storage
- **Access Control**: Only accessible by the App
- **Backup**: May be included in iCloud backups

### 4.2 Security Measures

- AES encryption for data protection
- HTTPS communication (Notion API, OpenBD API)
- Code obfuscation and signature verification

### 4.3 Data Deletion

- **App Uninstall**: All data is completely deleted
- **Clear History**: Can be deleted using the "Clear History" feature in the app
- **Token Deletion**: Authentication information can be cleared from settings

## 5. Third-Party Data Sharing

Except in the following cases:

## 7. California Residents' Rights (CCPA)

Under the California Consumer Privacy Act (CCPA), California residents have the following rights:

- **Right to Access**: Request disclosure of collected personal information
- **Right to Delete**: Request deletion of personal information
- **Right to Opt-Out of Sale**: Refuse sale of personal information (we do not sell)

**Note**: Since the App does not send personal information to servers, we do not hold any of your personal information.

---

## 8. European Residents' Rights (GDPR)

Under the EU General Data Protection Regulation (GDPR), European Economic Area (EEA) residents have the following rights:

- **Right to Access**: Request a copy of personal data
- **Right to Rectification**: Request correction of inaccurate personal data
- **Right to Erasure (Right to be Forgotten)**: Request deletion of personal data
- **Right to Restrict Processing**: Request restriction of personal data processing
- **Right to Data Portability**: Request transfer of personal data
- **Right to Object**: Object to personal data processing

**Note**: Since the App does not send personal data to servers, we do not hold any of your personal data. All data is stored on-device and completely deleted upon app uninstallation.
